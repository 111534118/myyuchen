新手指南：完成第一個主線任務

歡迎來到《Peak》的險峻世界！本攻略將帶你一步步完成遊戲的開端，讓你從一名新手順利成長為登山好手。

第一步：接受任務

進入遊戲後，前往位於「寧靜森林」的村莊中心，找到名為「長老」的 NPC。與他對話，他會給你第一個主線任務：「尋找失落的羅盤」。

第二步：前往低語洞穴

長老會提示你，羅盤最後出現的地點是村莊東南方的「低語洞穴」。打開你的地圖，朝著標示的方向前進。沿途你會遇到一些無害的小動物，不必理會。

第三步：取得羅盤並返回

進入洞穴後，你會在深處的一個石台上發現「失落的羅盤」。拾取它，然後原路返回村莊，將羅盤交給長老即可完成任務。恭喜你，完成了第一步！
